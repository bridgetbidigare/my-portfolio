My weather app →

The goal for this project was to create a weather app that could tell you the current weather (in fahrenheit or celsius) in any city, or with the user's current location. This project includes a 5-day forecast and embedded Spotify playlists that change depending on the weather. This was my first time integrating an API and it was so fun!

Made using: HTML5, CSS3, vanilla JavaScript, and API
